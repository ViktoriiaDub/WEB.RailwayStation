# УкрЗалізниця - Система бронювання квитків

Веб-застосунок для моделювання роботи системи продажу залізничних квитків (аналог Укрзалізниці), реалізований на React.

## Функціонал

-  Список потягів та рейсів
-  Пошук за маршрутом або номером потяга
-  Вибір вагона та місць
-  Форма бронювання з валідацією
-  Збереження бронювань у localStorage
-  Сповіщення про успішне бронювання

## Технічний стек

- **React 18** - компонентний підхід
- **Vite** - збірка проєкту
- **React Router DOM v6** - маршрутизація
- **CSS Modules** - стилізація

## Запуск проєкту

```bash
# Встановити залежності
npm install

# Запустити dev-сервер
npm run dev

# Збірка для продакшену
npm run build
```
